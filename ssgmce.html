Web Development Module Documentation for Fab Lab


Module Description: This module aims to equip participants with the skills and knowledge to design, develop, and deploy interactive and user-friendly websites for Fab Lab. Participants will learn the fundamentals of web development, including HTML, CSS, JavaScript, and relevant frameworks. The module will also cover best practices for creating websites that effectively communicate Fab Lab's projects, resources, and community engagement.


< Projects>

• Learning Objectives:


1. Understand the basics of web development and its importance in promoting Fab Lab's activities. 2. Learn HTML and CSS for structuring and styling web content. 3. Develop proficiency in JavaScript for creating interactive elements on web pages. 4. Explore responsive web design principles for optimal viewing on various devices. 5. Integrate multimedia elements to showcase Fab Lab projects effectively. 6. Implement user-centered design principles to enhance user experience. 7. Learn basics of version control and collaboration for web development projects.


• Assessment:

- Weekly coding assignments to practice concepts covered. - Mid-term project: Design a static web page highlighting a Fab Lab project. - Final project: Develop an interactive and responsive Fab Lab website. - Participation in peer evaluations and discussions.
